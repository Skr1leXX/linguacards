# 🧠 Spaced Repetition App

Веб-приложение для изучения иностранных слов с научно обоснованной системой интервального повторения (алгоритм Лейтнера).

## 🌟 Особенности

- **🎯 Алгоритм Лейтнера** - научный подход к запоминанию
- **📊 Детальная статистика** - графики и аналитика прогресса
- **📱 PWA** - работает как нативное приложение
- **🔐 Безопасность** - JWT аутентификация, хеширование паролей
- **🌍 Мультиязычность** - поддержка нескольких языков изучения
- **📱 Адаптивный дизайн** - для всех устройств

## 🚀 Быстрый старт

### Вариант 1: Docker (рекомендуется)
```bash
# 1. Клонируйте репозиторий
git clone <your-repo>
cd spaced-repetition

# 2. Настройте окружение
cp backend/.env.example backend/.env
# Отредактируйте backend/.env

# 3. Запустите через Docker Compose
docker-compose up --build

# Приложение доступно по адресу: http://localhost:5000
